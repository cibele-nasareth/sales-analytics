Projeto de análise de vendas desenvolvido em MySQL, simulando a operação de um e-commerce desde o registro de clientes e produtos até o processamento das vendas. A base de dados foi estruturada com relacionamentos entre tabelas para representar um cenário comercial real, permitindo a exploração de dados através de consultas SQL, análise de indicadores de desempenho e geração de insights sobre vendas, produtos, clientes e vendedores.
